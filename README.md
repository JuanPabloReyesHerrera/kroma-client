# ꓘROMA - Sistema de Reserva de Citas

Este es un ecosistema digital diseñado para la peluquería **ꓘROMA** en Bolivia.

## Tecnologías utilizadas:

- **React** (Frontend)
- **Tailwind CSS** (Estilos)
- **Supabase/Firebase** (Base de Datos - En proceso)

## Funcionalidades:

- Selección de sede, servicio y barbero.
- Gestión de horarios disponibles.
- Resumen de reserva y formulario de contacto.
